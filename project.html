Real-Time Notification Microservice

A fully asynchronous, multi-channel notification system built with Django, DRF, WebSockets, Celery, and Redis.

Features

  • REST API for sending & retrieving notifications
  • Channels-based WebSocket real-time delivery
  • Celery + Redis for background jobs
  • Multi-channel fallback (Email → SMS → WebSocket)
  • Delivery logs, retries, & scheduling
